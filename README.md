# Payroll Management Information System (PMIS)

A comprehensive **Payroll Management Information System (PMIS)** designed to streamline payroll operations, automate salary calculations, monitor employee attendance, manage payroll records, and provide interactive analytics for Human Resources (HR) and Finance departments.

This project combines **business process analysis**, **database design**, and **interactive dashboard development** to simulate a real-world enterprise payroll solution.

---

## Project Overview

Payroll processing is one of the most critical business functions in any organization. Manual payroll systems are often time-consuming, error-prone, and difficult to monitor.

The Payroll Management Information System (PMIS) was designed to improve payroll efficiency by automating payroll calculations, integrating attendance records, managing employee information, and providing decision-makers with interactive reports and dashboards.

This project was developed as part of the **Management Information Systems (MOIS)** course and demonstrates the complete system development process—from business analysis and process modeling to database design and dashboard development.

## Business Process

The system follows a structured monthly payroll workflow that includes:

1. Employee and attendance validation
2. Overtime approval
3. Bonus and commission calculation
4. Payroll calculation
5. Payroll approval
6. Bank transfer generation
7. Digital payslip generation
8. Reporting and dashboard updates

The complete workflow is documented using a **Swimlane Diagram** located in the **diagrams** folder.

---

## Database Design

The project includes a relational database that manages:

- Salary Components
- Overtime

The database structure is documented using an **Entity Relationship Diagram (ERD)**.
